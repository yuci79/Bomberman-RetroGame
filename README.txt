INSTALLATION:

- The game was developed under Ubuntu based Linux system. 
- The program is using SDL graphics libraries. It should be installed before running the program. see following web site for more instructions:
http://lazyfoo.net/SDL_tutorials/lesson01/linux/index.php

BUILD & EXECUTING:
- build command and all parameters is included in build.sh script. We should run it .
- Execute the game with following command on Terminal: 

	./BomberMan-RetroGame
